# 科研论文汇报 PPT 生成工具

将科研论文转化为简洁清晰的汇报 PPT，适用于组会汇报、课程展示、学术交流等场景。

## 功能特点

- 📄 **论文转 PPT**：从 PDF 论文自动生成 6-10 页的学术汇报幻灯片
- 🎨 **模板支持**：可提供参考 PPT 模板，自动提取配色、字体、布局风格
- 📝 **演讲稿支持**：支持 .docx / .pdf / .txt / .md 格式的学习笔记/演讲稿
- 🖼️ **双模式**：
  - **手动模式**：用户预先准备好图片，精确控制展示效果
  - **自动模式**：AI 从论文中提取图片，完全托管

## 快速开始

### 1. 准备材料

创建一个文件夹，放入以下文件：

```
项目文件夹/
├── paper.pdf        # 论文 PDF（必填）
├── images/          # 图片文件夹（手动模式必填）
│   ├── fig1_架构图.png
│   ├── fig2_实验结果.png
│   └── ...
├── outline.docx     # 演讲稿/学习笔记（可选）
└── template.pptx    # 参考模板（可选）
```

### 2. 使用方式

把文件夹路径发给 AI 助手，告诉它：
> "帮我把这篇论文做成汇报 PPT"

### 3. 获取结果

AI 会生成 `.pptx` 文件，包含：
- 封面（论文标题、作者）
- 问题背景
- 方法/模型架构
- 实验结果
- 创新点总结
- 局限与展望

## 手动模式（推荐）

手动模式下，你需要预先在 `images/` 文件夹中放入想要展示的图片：

**优点：**
- ✅ 精确控制图片裁剪位置
- ✅ 确保长宽比适合 PPT
- ✅ 图注完整不截断
- ✅ 避免 AI 提取图片不准的问题

**建议：**
- 图片分辨率 ≥ 1280x720
- 包含完整图注
- 文件命名清晰：fig1_架构图.png、fig2_结果对比.png

## 自动模式

如果没有准备 `images/` 文件夹，AI 会从论文 PDF 中自动提取图片。

**适用场景：**
- 想要完全托管，省心
- 论文图片质量较好、易于提取

## 检查清单

生成后检查：
- [ ] 论文标题是否正确？（避免同名论文混淆）
- [ ] 图片是否清晰完整？
- [ ] 文字是否超出页面边界？

## 依赖

- Python 3.8+
- python-pptx
- PyMuPDF (fitz)
- python-docx（处理 .docx 演讲稿）
